## ✍️ 写作指南

### 标题设计
- 吸引眼球但不夸大
- 包含关键词
- 字数控制在20字内

### 内容结构
```
标题
├── 引入（钩子）
├── 核心内容（干货）
├── 案例分析（证明）
└── 行动号召（转化）
```

### 文风特点
- 通俗易懂
- 有心理学视角
- 避免专业术语堆砌
- 加入个人观点和故事
