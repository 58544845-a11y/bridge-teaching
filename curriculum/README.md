## 📝 大纲模板

每门课程的大纲应包含：

1. **课程目标** - 学完后能做什么
2. **受众群体** - 适合什么水平的学员
3. **课程时长** - 预计课时
4. **主要内容** - 核心知识点
5. **练习设计** - 配套练习题
6. **考核方式** - 如何检验学习效果
