# 防守信号详解

## 概述

防守信号是同伴之间沟通的重要工具，帮助协调防守策略。

## 三大基本信号

### 1. 态度信号 (Attitude Signal)

**含义：** 表示对当前花色的态度

**用法：**
- **大牌（鼓励）**：表示喜欢当前花色，希望同伴继续攻
- **小牌（不鼓励）**：表示不喜欢当前花色，希望同伴换攻

**示例：**
```
同伴首攻♠A，你持♠J52
- 出♠J：鼓励，希望同伴继续攻♠
- 出♠2：不鼓励，希望同伴换攻
```

### 2. 计数信号 (Count Signal)

**含义：** 表示该花色的张数

**用法：**
- **先大后小**：偶数张（2、4、6）
- **先小后大**：奇数张（3、5、7）

**示例：**
```
你持♠KJ52（4张）
第一墩出♠K，第二墩出♠2 → 表示4张

你持♠KJ5（3张）
第一墩出♠5，第二墩出♠J → 表示3张
```

### 3. 花色选择信号 (Suit Preference Signal)

**含义：** 在无法赢墩时，表示希望换攻的花色

**用法：**
- **较大牌**：希望换攻高级花色（♠>♥>♦>♣）
- **较小牌**：希望换攻低级花色

**示例：**
```
同伴首攻♠A，你持♠KJ52 ♥AQJ ♦K3 ♣QJ4
希望同伴换攻♥（高级花色），出♠K
希望同伴换攻♣（低级花色），出♠2
```

## 具体应用场景

### 场景1：同伴首攻A

**情况：** 同伴首攻♠A，你有♠KJ5

**信号选择：**
- 有♠Q或希望同伴继续：出大牌（态度信号）
- 没有兴趣：出小牌（态度信号）
- 希望换攻其他套：花色选择信号

### 场景2：跟牌时

**情况：** 庄家出♥，你需要跟牌

**信号选择：**
- 保留大牌控制：出小牌（态度信号）
- 给同伴计数：计数信号

### 场景3：垫牌时

**情况：** 无法跟牌，需要垫牌

**信号选择：**
- 花色选择信号（最重要）
- 表示希望同伴换攻的套

## 高级信号技巧

### Smith信号

**定义：** 在明手摊牌后，第一墩的跟牌表示对该花色的态度

**用法：**
- 大=鼓励，小=不鼓励
- 用于首攻花色

### 反史密斯信号

**定义：** 与史密斯信号相反

**用法：**
- 某些叫牌体系使用
- 需要与同伴约定

###  Odd-Even信号

**定义：** 奇偶信号

**用法：**
- 奇数牌=鼓励/高级花色
- 偶数牌=不鼓励/低级花色

## 常见错误

### ❌ 错误1：信号混乱

**问题：** 同伴不理解你的信号含义

**解决：** 与同伴明确约定信号体系

### ❌ 错误2：过早暴露大牌

**问题：** 过早出大牌失去控制

**解决：** 谨慎使用态度信号

### ❌ 错误3：忽视花色选择

**问题：** 错过换攻良机

**解决：** 养成垫牌给信号的习惯
