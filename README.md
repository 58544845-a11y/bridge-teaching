## 🤝 协作方式

**课程开发：**
1. 在 `curriculum/` 编写课程大纲
2. 在 `lessons/` 编写逐课教案
3. 在 `materials/` 整理教学素材

**推广内容：**
1. 在 `promotion/` 创建文案
2. 在 `psychology/` 撰写心理学文章

**自动化工：**
- 使用 GitHub Actions 自动化处理
- 批改脚本、通知提醒等
