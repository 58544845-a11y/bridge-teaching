## 🎮 训练模块详解

### 模块一：心理战术 (Psychological Warfare)

**核心技巧：**
- 🎭 **假牌技巧** - 何时出假牌、如何迷惑庄家
- 🧠 **读牌技巧** - 从叫牌和出牌推断牌型
- 🎪 **心理首攻** - 非常规首攻的选择
- 🎨 **信号伪装** - 假信号的使用与识别

**示例题目：**
```
定约：4♠，你是西家
你的牌：♠KJ52 ♥AQ ♦KJ3 ♣QJ52

庄家出♥2，明手♥K，你是第二家...

[高级技巧] 出♥A还是♥Q？
- ♥A：立即赢墩，但暴露实力
- ♥Q：假牌迷惑，让庄家误判

正确答案：♥Q（假装没有♥A）
原理：保留♥A控制，迷惑庄家关于♥的分布
```

## 🚀 快速开始

### 安装运行

```bash
# 克隆项目
git clone https://github.com/yourusername/bridge-defense-elite.git

# 进入目录
cd bridge-defense-elite

# 安装依赖
npm install

# 启动训练
npm start
```

