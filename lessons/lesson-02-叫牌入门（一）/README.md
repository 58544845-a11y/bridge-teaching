### 第一部分：导入（10分钟）

#### 1.1 回顾上节课

**提问互动：**
> "上节课我们打了什么？"
>
> 答案：Mini-Bridge（迷你桥牌）
>
> "Mini-Bridge 和真实桥牌有什么区别？"
>
> 答案：Mini-Bridge 不叫牌，直接打牌

#### 1.2 为什么需要叫牌？

**问题引入：**
> "如果没有叫牌，我们怎么知道定约是什么？"
>
> "定约是什么？"
> "怎么判断谁的牌更好？"

**叫牌的意义：**

```
叫牌就是：谈判协商的过程

目的：
1. 确定定约（打什么花色？几级？）
2. 评估实力（谁牌好？谁牌差？）
3. 传递信息（告诉搭档我有什么）
4. 最终决策（最终定约是什么）
```

**类比：**
```
叫牌像什么？
   - 像买房谈判：出价、还价、协商
   - 像公司投标：评估实力、决定投标
   - 像团队分工：讨论谁负责什么
```

### 第二部分：点数计算（35分钟）

#### 2.1 什么是点数？

**基本概念：**
- **点数** 是牌力的量化指标
- 一手牌的点数 = 每张大牌的点数之和
- 点数越高，牌力越强

#### 2.2 如何计算点数？

**大牌点数表：**

```
牌面    点数
───────────────
A       4点
K       3点
Q       2点
J       1点
10-2    0点
```

**记忆口诀：**
> "A四，K三，Q两，J一，其他都没用！"

---

#### 2.3 点数计算练习

**示例1：简单点数计算**

```
手牌：
  ♠: A K Q J
  ♥: 5 4
  ♦: 8 7
  ♣: 6 5 3

计算过程：
  ♠: A(4) + K(3) + Q(2) + J(1) = 10点
  ♥: 0点
  ♦: 0点
  ♣: 0点
  ─────────────────
  总计：10点
```

---

**示例2：完整点数计算**

```
手牌：
  ♠: A Q 8
  ♥: K J 7
  ♦: Q 10 5 2
  ♣: A K 3

计算过程：
  ♠: A(4) + Q(2) = 6点
  ♥: K(3) + J(1) = 4点
  ♦: Q(2) = 2点
  ♣: A(4) + K(3) = 7点
  ─────────────────
  总计：19点
```

---

**互动练习：**

给学员发牌，让他们计算点数。

```
练习牌1：
  ♠: K 8 4
  ♥: A Q 7
  ♦: J 9 5 2
  ♣: Q 6 3

你的点数：_____点

（答案：3 + 4+2 + 1 + 2 = 12点）
```

#### 2.4 点数与牌力

**点数等级表：**

```
点数范围      牌力等级
────────────────────────
0-5点        极弱
6-9点        较弱
10-12点      中等
13-15点      较强
```

### 第三部分：开叫规则（30分钟）

#### 3.1 什么是开叫？

**定义：**
- **开叫** 是叫牌过程中第一个叫出花色或无将的人
- 开叫者通常是牌力最强的人

**叫牌顺序：**
```
发牌人 → 西家 → 北家 → 东家 → ...
              ↓
          第一个叫牌的人就是"开叫者"
```

---

#### 3.2 什么时候应该开叫？

**基本规则：**

```
✅ 应该开叫的条件：
   - 点数 ≥ 13点
   - 有一个好的花色（至少4-5张）
   - 或者牌型特别好

❌ 不应该开叫的条件：
   - 点数 < 12点
   - 牌型太平（没有强花色）
   - 对手牌太强（判断）
```

---

#### 3.3 开叫的牌力标准

**13点原则：**

```
点数          建议
────────────────────
0-11点        不开叫（PASS）
12-15点       轻开叫（边缘牌）
16-18点       正常开叫
19-21点       强开叫
22+点         2♣开叫（强牌）
```

---

#### 3.4 开叫的选择

**开叫的花色选择：**

1. **高花优先**（♠️、♥️）
   - 黑桃有5张以上 → 开叫 1♠
   - 红心有5张以上 → 开叫 1♥

2. **低花开叫**（♦️、♣️）
   - 没有高花5张 → 开叫低花
   - 通常选择较长的低花
   - 如果一样长，选择 ♦️

3. **无将开叫**（NT）
   - 均型牌（没有特别长的花色）
   - 15-17点 → 开叫 1NT

---

#### 3.5 叫牌术语

**基本术语：**

| 术语 | 含义 | 示例 |
|------|------|------|
| **开叫** | 第一个叫牌 | 南叫 1♠ |
| **应叫** | 搭档回应叫牌 | 北叫 2♥ |
| **加叫** | 叫高一阶的花色 | 1♠ → 2♠ |
| **跳叫** | 叫高一阶以上 | 1♠ → 3♠ |
| **Pass** | 不叫 | Pass |
| **加倍** | 惩罚性加倍 | X |
| **再加倍** | 取消加倍 | XX |

### 第五部分：战局复盘（10分钟）

#### 5.1 常见错误分析

**错误1：忘记算点数**

```
错误示范：
  拿牌就开叫，不计算点数

正确示范：
  先计算点数，再决定是否开叫
  "我算了点数，18点，可以开叫！"
```

---

**错误2：边缘牌判断失误**

```
错误示范：
  12点就开叫（太激进了）
  或者 15点不叫（太保守了）

正确示范：
  12-15点是边缘牌，需要结合牌型判断
  "12点但有好的花色，可以考虑开叫"
```

---

**错误3：忽略牌型**

```
错误示范：
  只看点数，不看花色长度

正确示范：
  牌型和点数同样重要
  "15点但花色分散，可能不如13点加好花色"
```

## 🎯 教学提示

### 心理关注

- **认知负荷**：点数计算是新概念，避免一次性给太多信息
- **焦虑管理**：边缘牌判断时，强调"没有绝对正确"
- **成就感**：每算对一次点数都及时表扬

### 教学技巧

- **口诀记忆**：用口诀帮助记忆点数
- **实物演示**：用真实扑克牌展示
- **分组练习**：让学员两两互助

### 应急处理

- **学不会怎么办？** → 一对一辅导，多给例子
- **觉得太难怎么办？** → 调整节奏，多练习少讲解
- **出错很多怎么办？** → 耐心纠正，强调错误是学习过程
