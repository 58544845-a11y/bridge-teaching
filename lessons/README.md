```
lessons/
```

---

## 📝 教案模板

### 课时概述 (README.md)
```markdown
# 第X课：[课程名称]

**时长：** 90分钟
**难度：** ⭐⭐⭐
**先修课程：** 无

## 学习目标
- [ ] 理解基本概念
- [ ] 掌握核心技能
- [ ] 完成练习题

## 教学重点
- 核心知识点1
- 核心知识点2

## 时间安排
| 环节 | 时长 | 内容 |
|------|------|------|
| 导入 | 10min | 引入主题 |
| 讲解 | 40min | 核心内容 |
| 练习 | 30min | 互动练习 |
| 总结 | 10min | 知识梳理 |
```

### 主要内容 (content.md)
详细的教学内容、知识点讲解

### 案例分析 (examples.md)
实际牌例分析、错误示范

### 练习题 (exercises.md)
本节课的配套练习题

---

## 📌 注意事项

1. **循序渐进** - 确保学员能跟上节奏
2. **案例驱动** - 多用实际牌例
3. **互动练习** - 让学员动手操作
4. **心理关注** - 注意学员情绪和反馈
