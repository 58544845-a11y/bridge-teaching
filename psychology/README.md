## 📝 文章格式

每篇文章应包含：

1. **问题引入** - 从实际教学场景切入
2. **理论背景** - 简述相关心理学理论
3. **桥牌应用** - 理论在桥牌中的具体体现
4. **教学建议** - 对课程设计的启示
5. **案例支持** - 真实案例说明

---

## 🎯 核心理念

### 认知视角
- 桥牌是复杂认知任务
- 需要工作记忆、长时记忆、执行功能
- 每个人的认知风格不同

### 情绪视角
- 桥牌中的胜负情绪
- 学习过程中的挫折感
- 群体竞争与合作

### 社会视角
- 桥牌是社交游戏
- 沟通与信号
- 建立信任关系
