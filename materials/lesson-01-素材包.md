## 🎴 牌型对比练习（第2节用）

### 练习1：牌的大小排序

**问题：** 把这几张牌按从大到小排序

```
[ ] [ ] [ ] [ ] [ ]
  ↓   ↓   ↓   ↓   ↓
♠️7  ♥️K  ♣️A  ♦️Q  ♠️J
```

**答案：**
```
♣️A  >  ♥️K  >  ♠️J  >  ♦️Q  >  ♠️7
```

## 🎯 常见错误对照

### 错误1：不跟牌

```
错误示范：
  南出♠5
  西出♥K  ← 错误！必须跟同花色
  ...

正确示范：
  南出♠5
  西出♠8  ← 正确！跟了同花色
  ...
```

### 错误2：忘了明手可以看

```
提醒学员：
  "北家的牌是明手，摊开在桌上！
   南家（庄家）可以随时查看明手的牌，
   这是我们很大的优势！"
```

### 错误3：出牌前不思考

```
教学提示：
  "出牌前先问自己三个问题：
   1. 我现在要出什么花色？
   2. 我有什么牌？
   3. 哪张牌最有可能赢？"
```

```
出牌顺序：
  1. 每个人按顺序出牌
  2. 必须跟同花色
  3. 同花色最大的赢这墩
  4. 赢家下一轮先出牌
```
