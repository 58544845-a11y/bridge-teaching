# 教学素材

本目录存放各类教学资源。

---

## 📂 素材分类

```
materials/
├── images/          # 图片素材
├── diagrams/        # 牌例图表
├── handouts/        # 讲义打印稿
├── slides/          # PPT 幻灯片
└── videos/          # 视频素材
```

---

## 🖼️ 图片素材

- 牌型示意图
- 叫牌表
- 概念图解

---

## 📊 牌例图表

- 标准牌例格式
- 特殊情况标注
- 分析过程可视化

---

## 📄 讲义打印稿

每节课的可打印讲义，方便学员课后复习。

---

## 🎯 使用指南

1. **图片素材** - 可直接在文档中引用
2. **牌例图表** - 使用 markdown 表格或代码块展示
3. **讲义** - 导出为 PDF 分发
4. **PPT** - 课堂演示使用
