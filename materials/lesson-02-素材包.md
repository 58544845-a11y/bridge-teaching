```

────────────────────────
0-11点        PASS（不开叫）

16-18点       正常开叫
19-21点       强开叫
22+点         2♣开叫（超强）
```

---

### 花色选择指南

```
1. 高花优先（♠️、♥️）
   - 黑桃5张以上 → 开叫 1♠
   - 红心5张以上 → 开叫 1♥

2. 低花选择（♦️、♣️）
   - 没有高花5张 → 开叫低花
   - 选择较长的低花
   - 一样长选 ♦️

3. 无将开叫（NT）
   - 均型牌（无特别长花色）
   - 15-17点 → 开叫 1NT
```

## 🎯 常见问题解答（FAQ）

### Q1：12点应该开叫吗？

**A：** 12点是边缘牌，需要结合牌型判断：
- 有好的花色（5张以上）→ 可以开叫
- 牌型很平 → 通常PASS
- 对手牌太强 → 慎重考虑

---

### Q2：什么是均型牌？

**A：** 没有特别长花色的牌：
- 每个花色3-4张
- 或者 4-3-3-3、4-4-3-2 等均衡分布
- 均型牌适合开叫无将

---

### Q3：为什么高花优先？

**A：** 高花定约更容易：
- 高花只需要8墩就能成局（4♠/4♥）
- 低花需要11墩（5♦/5♣）
- 所以高花更有价值

---

### Q4：开叫后对手加倍怎么办？

**A：** 这是惩罚性加倍：
- 对手认为你的开叫不合理
- 如果牌力确实不够，可以选择再叫或PASS
- 但初学者暂时不用太担心，正常开叫就行
