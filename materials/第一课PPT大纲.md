## 第15页：常见错误

**标题：** 小心这些错误

**错误1：不跟牌**
```
❌ 南出♠️5，西出♥️K
✅ 南出♠️5，西出♠️8
```

**错误2：忘了明手**
```
💡 北家的牌是公开的！
💡 利用明手的优势
```

**错误3：出牌前不思考**
```
💭 出什么花色？
💭 我有什么牌？
💭 哪张可能赢？
```

## 📋 PPT制作建议

### 视觉风格

**配色方案：**
- 背景：白色或浅米色
- 标题：深蓝色 (#1a365d)
- 正文：深灰色 (#2d3748)
- 强调：桥牌花色色
  - 黑桃/梅花：黑色
  - 红心/方块：红色 (#e53e3e)

**字体建议：**
- 标题：思源黑体 Bold / Arial Bold
- 正文：思源黑体 Regular / Arial Regular
- 字号：标题 32-40pt，正文 18-24pt

**图标使用：**
- 使用简洁的图标
- 桥牌相关符号
- 避免过于花哨

---

### 动画建议

**适当使用：**
- 淡入淡出（内容出现）
- 擦除（列表项）
- 放大（重点强调）

**避免：**
- 过于复杂的动画
- 音效
- 过多的动画效果

---

### 备注使用

**每页备注：**
- 讲解要点
- 时间提示
- 互动提示
- 注意事项

**示例（第2页备注）：**
```
时间：2分钟
要点：
- 强调"体验乐趣"最重要
- 不是要记住所有规则
互动：
- 问学员"你最期待学什么？"
```
